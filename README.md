# MindScale — דף נחיתה

דף נחיתה ללכידת לידים לסוכנות **MindScale** — מודל ה-WIN-WIN היחיד בארץ.

## 🔗 קישורים
- **דף חי (Production):** https://mindscale-landing.vercel.app
- **דשבורד Vercel:** https://vercel.com/yossi-shakarof/mindscale-landing

## 📁 מבנה
- `index.html` — הדף כולו בקובץ אחד (HTML + CSS + JS, ללא תלויות). פשוט פותחים בדפדפן.

## ✏️ איך עורכים
1. פותחים את `index.html` בכל עורך (VS Code מומלץ) או ישירות בדפדפן.
2. הדף כולל **עורך טקסט אינליין** (הכפתורים ✏️ / 💾 / ↺ בפינה השמאלית-תחתונה) לשינויי טקסט מהירים בתצוגה.
   > הערה: שינויים דרך העורך האינליין נשמרים ב-localStorage של הדפדפן בלבד ולא מסתנכרנים. לשינוי קבוע — עורכים את `index.html`.
3. שינויים אמיתיים בקוד → `git commit` + `git push` → Vercel מעלה אוטומטית.
